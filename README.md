# 🐾 Pet Collar System

## 📌 Overview

A smart IoT-based pet collar system designed to monitor and track pets in real-time.
The system integrates embedded hardware with a mobile application to provide live data such as location, activity, and health indicators.

---

## 🚀 Features

* 📍 Real-time pet tracking
* ❤️ Health monitoring (heart rate / activity)
* 📱 Mobile app (Flutter) for visualization
* 🔗 Bluetooth Low Energy (BLE) communication
* 🔔 Alerts & notifications



## 🛠️ Tech Stack

### 🔹 Hardware

* ESP32 / ESP32-C3
* Sensors (Heart rate, Motion, etc.)
* Buzzer (optional features)

### 🔹 Software

* Flutter (Mobile App)
* Arduino (Embedded Programming)
* BLE Communication



## ⚙️ How It Works

1. Sensors collect data from the pet.
2. ESP32 processes and sends data via BLE.
3. Flutter app receives and displays data in real-time.
4. User monitors pet status through the app.

## ▶️ Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/yourusername/pet-collar-system.git
```

### 2. Run Flutter App

```bash
cd mobile
flutter pub get
flutter run
```

### 3. Upload Arduino Code

* Open Arduino IDE
* Select ESP32 board
* Upload code from `hardware/`
